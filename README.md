A modern, responsive chat interface inspired by Claude/ChatGPT.  
Built with HTML, CSS, JavaScript, jQuery, and Bootstrap 5.

## How to Run

1. Unzip `Kowshik_BH_ChatUI.zip`
2. Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge)
3. No server or build step needed — it works straight from the file system

## Features Implemented

- Welcome screen with 4 clickable suggestion cards
- User and AI message bubbles with avatars, names, timestamps
- Animated typing indicator (bouncing dots)
- Auto-resizing textarea input
- Send button disabled when input is empty
- Enter to send, Shift+Enter for new line
- Mock AI responses with 1–2 s simulated delay
- Sidebar with New Chat button and chat history
- Mobile-responsive with hamburger menu + overlay
- Smooth fade/slide animations throughout
- Custom scrollbar styling (Bonus)
- CSS variables for easy theming
